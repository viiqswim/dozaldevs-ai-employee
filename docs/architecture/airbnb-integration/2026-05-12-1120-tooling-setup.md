## Notes

- Homebrew's mitmproxy 10.0.0 install was broken (missing cryptography module with version conflict)
- Resolved by installing fresh via `uv tool install mitmproxy` → installs 11.0.2
- Python: 3.11.11, OpenSSL: 3.4.1

## Quick Start

```bash
# Start mitmweb (web UI proxy)
mitmweb --listen-host 0.0.0.0 --listen-port 8080

# Start mitmdump (capture to file)
mitmdump -w /tmp/airbnb-research/captures/airbnb-session.flow

# Generate OpenAPI spec from HAR
mitmproxy2swagger -i /tmp/airbnb-research/captures/airbnb-web-session.har \
  -o /tmp/airbnb-research/openapi-spec-raw.yaml \
  -p https://www.airbnb.com/api

# Generate OpenAPI spec from mitmproxy flow file
mitmproxy2swagger -i /tmp/airbnb-research/captures/airbnb-session.flow \
  -o /tmp/airbnb-research/openapi-spec-raw.yaml \
  -p https://www.airbnb.com/api \
  -f flow
```

## Browser DevTools (HAR) Workflow

1. Open Chrome → airbnb.com → login
2. F12 → Network tab → check "Preserve log"
3. Filter: `Fetch/XHR` or `All`
4. Perform target actions (inbox, reservations, calendar, listing)
5. Right-click any request → Save all as HAR with content
6. Save to: `/tmp/airbnb-research/captures/airbnb-web-session.har`
7. Run mitmproxy2swagger on the HAR file (first pass)
8. Edit YAML: remove `ignore:` prefix from relevant paths
9. Run second pass with same command to generate full descriptions
