## Current Tech Stack

| Component | Technology |
|---|---|
| Orchestration + Queue | Inngest (managed durable workflows) |
| Engineering agent runtime | OpenCode (`opencode serve` + SDK) on Fly.io |
| Non-engineering agent runtime | Inngest in-process TypeScript |
| Database + State | Supabase (PostgreSQL + pgvector) |
| LLM Gateway | OpenRouter (unified API for 100+ models) |
| LLM Cost Optimization | Claude Max 20x subscription |
| Execution Compute (Engineering) | Fly.io Machines API |
