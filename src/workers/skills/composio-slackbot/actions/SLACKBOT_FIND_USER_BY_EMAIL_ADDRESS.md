# SLACKBOT_FIND_USER_BY_EMAIL_ADDRESS

**Description**: Retrieves the Slack user object for an active user by their registered email address; requires the users:read.email OAuth scope. Fails with 'users_not_found' if the email is unregistered, the user is inactive, the account is a guest, or the email is hidden by workspace privacy settings.
