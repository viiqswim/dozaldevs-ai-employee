# SLACKBOT_RETRIEVE_USER_PROFILE_INFORMATION

**Description**: Retrieves profile information for a specified Slack user (defaults to the authenticated user if `user` ID is omitted); a provided `user` ID must be valid. Sensitive fields like email and phone may be silently omitted if required scopes (e.g., `users:read.email`) are not granted or workspace privacy policies restrict access.

| description | unknown | No |  |
