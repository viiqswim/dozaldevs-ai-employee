# ELEVENLABS_ADD_PRONUNCIATION_DICTIONARY_FROM_RULES

**Description**: Tool to create a new pronunciation dictionary from provided rules for ElevenLabs text-to-speech. Use when you need to define custom pronunciations for specific words or phrases to improve speech accuracy. Supports both alias rules (simple text replacements) and phoneme rules (IPA phonetic notation).
