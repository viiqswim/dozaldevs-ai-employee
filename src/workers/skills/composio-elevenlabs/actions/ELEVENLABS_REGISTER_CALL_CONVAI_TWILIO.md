# ELEVENLABS_REGISTER_CALL_CONVAI_TWILIO

**Description**: Tool to register a Twilio call and return TwiML to connect the call to an ElevenLabs Conversational AI agent. Use when you need to integrate ElevenLabs agents with your own Twilio infrastructure for inbound or outbound calls.
