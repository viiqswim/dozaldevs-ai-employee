# ELEVENLABS_ATTACH_PHONE_NUMBER_TO_AGENT

**Description**: Tool to assign or unassign an existing imported phone number to an agent by updating the phone number's assigned agent. Use when you need to programmatically route inbound calls to a specific agent or change agent assignment for a phone number.
