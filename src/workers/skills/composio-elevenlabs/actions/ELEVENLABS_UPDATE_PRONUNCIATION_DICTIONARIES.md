# ELEVENLABS_UPDATE_PRONUNCIATION_DICTIONARIES

**Description**: Updates a project's pronunciation dictionaries on ElevenLabs to improve text-to-speech accuracy for specialized terms; note that while multiple dictionaries can be applied, the UI only displays the first.
