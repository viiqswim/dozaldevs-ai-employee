# ELEVENLABS_CREATE_CONVAI_KNOWLEDGE_BASE_RAG_INDEX

**Description**: Tool to compute or retrieve RAG indexes for multiple knowledge base documents in batch. Use when you need to enable semantic search on multiple knowledge base documents for conversational AI agents. This endpoint creates embeddings if missing or returns existing indexes, supporting up to 100 documents per request.
