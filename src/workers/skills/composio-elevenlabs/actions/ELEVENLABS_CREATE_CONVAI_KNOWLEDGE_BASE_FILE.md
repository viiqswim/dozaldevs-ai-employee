# ELEVENLABS_CREATE_CONVAI_KNOWLEDGE_BASE_FILE

**Description**: Tool to create a knowledge base document from an uploaded file for ElevenLabs Conversational AI agents. Use when you need to provide documentation or reference material that agents can access during conversations. The uploaded file content will be indexed and made available to agents for answering user queries.
