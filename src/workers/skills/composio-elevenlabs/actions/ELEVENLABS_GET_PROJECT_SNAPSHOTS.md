# ELEVENLABS_GET_PROJECT_SNAPSHOTS

**Description**: Retrieves all available snapshots (saved states or versions) for an existing project, enabling history tracking, version comparison, or accessing specific states for playback/processing, particularly in text-to-speech workflows.
