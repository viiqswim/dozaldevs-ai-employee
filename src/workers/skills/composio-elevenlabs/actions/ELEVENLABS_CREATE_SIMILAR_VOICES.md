# ELEVENLABS_CREATE_SIMILAR_VOICES

**Description**: Returns a list of shared voices similar to the provided audio sample. Use when you need to find voice matches based on audio characteristics. If neither similarity_threshold nor top_k is provided, default values will be applied.
