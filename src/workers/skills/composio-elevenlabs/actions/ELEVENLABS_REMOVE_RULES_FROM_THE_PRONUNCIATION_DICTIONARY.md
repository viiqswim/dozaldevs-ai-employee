# ELEVENLABS_REMOVE_RULES_FROM_THE_PRONUNCIATION_DICTIONARY

**Description**: Permanently removes exact-match pronunciation rules from a specified ElevenLabs pronunciation dictionary using a list of rule strings; non-matching rule strings are ignored and this action cannot add or modify rules.
