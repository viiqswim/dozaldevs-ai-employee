# ELEVENLABS_GET_DEFAULT_VOICE_SETTINGS

**Description**: Retrieves the ElevenLabs text-to-speech service's default voice settings (stability, similarity boost, style, speaker boost) that are applied when no voice-specific or request-specific settings are provided.
