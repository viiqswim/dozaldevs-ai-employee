# ELEVENLABS_DELETE_CONVAI_KNOWLEDGE_BASE_RAG_INDEX

**Description**: Tool to delete RAG index for a knowledge base document. Use when you need to remove semantic search indexing from a document to free up resources or when the document no longer needs to be indexed.
