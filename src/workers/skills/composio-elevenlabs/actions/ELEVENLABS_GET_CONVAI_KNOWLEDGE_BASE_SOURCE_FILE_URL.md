# ELEVENLABS_GET_CONVAI_KNOWLEDGE_BASE_SOURCE_FILE_URL

**Description**: Tool to get a signed URL to download the original source file of a file-type document from the knowledge base. Use when you need to download the original file that was uploaded to create a knowledge base document.
