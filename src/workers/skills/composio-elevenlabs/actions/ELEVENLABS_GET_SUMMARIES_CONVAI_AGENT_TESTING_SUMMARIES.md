# ELEVENLABS_GET_SUMMARIES_CONVAI_AGENT_TESTING_SUMMARIES

**Description**: Tool to retrieve multiple agent response test summaries by their IDs. Use when you need to fetch summary information for multiple tests in a single request. Returns a dictionary mapping test IDs to their summary details including name, type, and timestamps.
