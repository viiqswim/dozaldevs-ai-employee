# ELEVENLABS_UPDATE_AUDIO_NATIVE_CONTENT

**Description**: Tool to update content for an Audio Native project by uploading a text or HTML file. Use when you need to refresh or change the content of an existing Audio Native project. Optionally auto-convert to audio and auto-publish the new snapshot.
