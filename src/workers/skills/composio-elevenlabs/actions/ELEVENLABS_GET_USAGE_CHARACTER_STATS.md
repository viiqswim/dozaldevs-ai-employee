# ELEVENLABS_GET_USAGE_CHARACTER_STATS

**Description**: Tool to retrieve usage metrics for the current user or entire workspace. Returns time-series data with configurable aggregation intervals and breakdown types. Use when analyzing character usage patterns, tracking consumption by voice, user, or other dimensions over time.
