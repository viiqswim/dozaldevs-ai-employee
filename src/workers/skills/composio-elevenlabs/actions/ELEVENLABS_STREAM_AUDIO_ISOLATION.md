# ELEVENLABS_STREAM_AUDIO_ISOLATION

**Description**: Tool to remove background noise from audio and stream the isolated result. Use when you need to clean up audio recordings by isolating vocals/speech and removing unwanted background noise. The audio file must be at least 4.6 seconds long.
