# ELEVENLABS_CREATE_CONVERSATIONAL_AGENT

**Description**: Tool to create a new ElevenLabs Conversational AI agent with specified configuration. Use when you need to programmatically set up an agent with custom prompts, voice settings, and behavior. After creating the agent, you can chain other tools to attach phone numbers or configure additional settings.
