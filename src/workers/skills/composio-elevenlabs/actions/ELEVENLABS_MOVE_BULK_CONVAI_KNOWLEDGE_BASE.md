# ELEVENLABS_MOVE_BULK_CONVAI_KNOWLEDGE_BASE

**Description**: Tool to move multiple documents or folders from one folder to another in the knowledge base. Use when you need to reorganize multiple knowledge base items at once by moving them to a different folder or to the root level.
