# ELEVENLABS_CREATE_WORKSPACE_WEBHOOK

**Description**: Tool to create a new webhook for the workspace with specified authentication type. Use when you need to configure a webhook endpoint to receive notifications from ElevenLabs. The webhook_secret returned should be stored securely for verifying webhook authenticity.
