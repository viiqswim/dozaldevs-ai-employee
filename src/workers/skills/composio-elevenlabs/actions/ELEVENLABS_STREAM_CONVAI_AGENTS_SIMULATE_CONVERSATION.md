# ELEVENLABS_STREAM_CONVAI_AGENTS_SIMULATE_CONVERSATION

**Description**: Tool to run a simulated conversation between an agent and a simulated user, streaming back the response. Response is streamed as partial lists of messages that should be concatenated, with a final message containing conversation analysis once complete.
