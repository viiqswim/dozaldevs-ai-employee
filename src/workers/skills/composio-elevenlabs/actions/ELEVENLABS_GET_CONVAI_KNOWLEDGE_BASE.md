# ELEVENLABS_GET_CONVAI_KNOWLEDGE_BASE

**Description**: Tool to retrieve a list of available knowledge base documents. Use when you need to browse, search, or filter knowledge base documents by name, type, folder, or ownership. Supports pagination via cursor for large result sets.

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|

| title | unknown | No |  |
| properties | unknown | No |  |
