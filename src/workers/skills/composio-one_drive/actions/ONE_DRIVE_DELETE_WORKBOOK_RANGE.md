# ONE_DRIVE_DELETE_WORKBOOK_RANGE

**Description**: Deletes a range of cells from a worksheet in an Excel workbook stored in OneDrive. Use this action when you need to remove a specific cell range and shift the remaining cells to fill the gap. This action is irreversible — the deleted range cannot be recovered once removed. The operation requires Files.ReadWrite scope and the workbook must not be locked by another user or process.
