# ONE_DRIVE_CLEAR_WORKBOOK_RANGE

**Description**: Clears values, formats, or both from a worksheet range in an Excel workbook stored in OneDrive or SharePoint without deleting cells or shifting neighboring data. Use this action when you need to reset a range while preserving the worksheet's cell structure.
