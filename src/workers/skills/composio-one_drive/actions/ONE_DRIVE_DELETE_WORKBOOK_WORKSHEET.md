# ONE_DRIVE_DELETE_WORKBOOK_WORKSHEET

**Description**: Deletes a worksheet from a workbook in OneDrive by its ID or name. Use this action when you need to remove a specific worksheet from an Excel workbook stored in OneDrive or SharePoint. This action is irreversible — the worksheet cannot be recovered once deleted. Note that deleting the last visible worksheet in a workbook may result in an error.

|-----------|------|----------|-------------|
