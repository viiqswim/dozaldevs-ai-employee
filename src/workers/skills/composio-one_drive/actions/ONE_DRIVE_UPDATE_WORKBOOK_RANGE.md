# ONE_DRIVE_UPDATE_WORKBOOK_RANGE

**Description**: Updates a range in a workbook worksheet using Microsoft Graph API. Use this action when you need to write data, formulas, or number formats to specific cells in an Excel workbook stored in OneDrive or SharePoint. The address parameter supports various formats including single cells, cell ranges, entire columns, or entire rows.
