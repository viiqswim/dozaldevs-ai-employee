# GOOGLESHEETS_SET_DATA_VALIDATION_RULE

**Description**: Tool to set or clear data validation rules (including dropdowns) on a range in Google Sheets. Use when you need to apply dropdown lists, range-based dropdowns, or custom formula validation to cells.
