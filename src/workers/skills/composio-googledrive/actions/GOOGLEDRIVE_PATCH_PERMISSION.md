# GOOGLEDRIVE_PATCH_PERMISSION

**Description**: Tool to update a permission using patch semantics. Use when you need to modify specific fields of an existing permission without affecting other fields. **Warning:** Concurrent permissions operations on the same file are not supported; only the last update is applied.
