# GOOGLEDRIVE_LIST_ACCESS_PROPOSALS

**Description**: Tool to list pending access proposals on a file. Use when you need to retrieve access proposals for a specific file. Note: Only approvers can list access proposals; non-approvers will receive a 403 error.
