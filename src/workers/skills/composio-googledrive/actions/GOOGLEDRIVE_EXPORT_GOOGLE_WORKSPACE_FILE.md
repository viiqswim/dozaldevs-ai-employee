# GOOGLEDRIVE_EXPORT_GOOGLE_WORKSPACE_FILE

**Description**: Exports a Google Workspace document to the requested MIME type and returns exported file content. Use when you need to export Google Docs, Sheets, Slides, Drawings, or Apps Script files to a specific format. Note: The exported content is limited to 10MB by Google Drive API.
