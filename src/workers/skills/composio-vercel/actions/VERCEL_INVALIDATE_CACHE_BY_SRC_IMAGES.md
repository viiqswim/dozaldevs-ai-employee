# VERCEL_INVALIDATE_CACHE_BY_SRC_IMAGES

**Description**: Tool to invalidate edge cache by source image URLs. Use when you need to mark cached images as stale for specific source images. Invalidated images are revalidated in the background on the next request, ensuring zero latency impact for users.
