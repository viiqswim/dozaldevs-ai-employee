# SLACK_RETRIEVE_CONVERSATION_MEMBERS_LIST

**Description**: Retrieves a paginated list of active member IDs (not names, emails, or presence) for a specified Slack public channel, private channel, DM, or MPIM. Returns only user IDs; use a user-lookup tool to enrich member data.
