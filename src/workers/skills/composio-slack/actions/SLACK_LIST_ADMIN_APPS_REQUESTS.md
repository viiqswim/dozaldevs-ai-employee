# SLACK_LIST_ADMIN_APPS_REQUESTS

**Description**: Tool to list pending app installation requests for a team/workspace. Use when you need to see which apps users have requested to install that haven't yet been approved or denied. Requires Enterprise Grid or Business+ plan with admin.apps:read scope.
