# SLACK_REMOVE_REACTION_FROM_ITEM

**Description**: Removes an emoji reaction from a message, file, or file comment in Slack. Provide exactly one targeting method: channel+timestamp together, file, or file_comment. Mixing methods or omitting all returns invalid_arguments.
