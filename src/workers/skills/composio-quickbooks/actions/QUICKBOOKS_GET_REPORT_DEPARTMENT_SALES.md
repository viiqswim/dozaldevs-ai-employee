# QUICKBOOKS_GET_REPORT_DEPARTMENT_SALES

**Description**: Generate a department sales report in QuickBooks showing sales data broken down by department. Use when you need to analyze sales performance across different departments within a date range. Note: Department tracking must be enabled in the QuickBooks company file for this report to return data.
