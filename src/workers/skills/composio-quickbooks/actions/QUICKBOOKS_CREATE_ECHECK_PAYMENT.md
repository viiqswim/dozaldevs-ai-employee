# QUICKBOOKS_CREATE_ECHECK_PAYMENT

**Description**: Tool to create an eCheck payment using ACH bank transfer in QuickBooks Payments API. Use when processing payments from customer bank accounts via electronic check. The payment is debited directly from the customer's bank account using the provided routing and account numbers.
