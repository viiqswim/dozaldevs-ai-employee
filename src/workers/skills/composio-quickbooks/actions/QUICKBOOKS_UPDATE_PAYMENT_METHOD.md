# QUICKBOOKS_UPDATE_PAYMENT_METHOD

**Description**: Tool to update an existing payment method in QuickBooks Online. Use when you need to modify the name, status, or type of a payment method. Requires the payment method Id and current SyncToken.

|-----------|------|----------|-------------|

| title | unknown | No |  |
| required | unknown | No |  |
| properties | unknown | No |  |
| description | unknown | No |  |
