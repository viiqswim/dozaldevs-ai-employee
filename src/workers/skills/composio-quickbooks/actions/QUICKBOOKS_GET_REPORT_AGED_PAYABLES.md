# QUICKBOOKS_GET_REPORT_AGED_PAYABLES

**Description**: Retrieve an Aged Payables aging summary report from QuickBooks showing outstanding vendor balances grouped by age. Shows amounts owed to vendors categorized by aging periods (e.g., current, 1-30 days, 31-60 days overdue).
