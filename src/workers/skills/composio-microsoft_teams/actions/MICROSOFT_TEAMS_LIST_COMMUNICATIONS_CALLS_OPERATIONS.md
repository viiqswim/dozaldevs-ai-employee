# MICROSOFT_TEAMS_LIST_COMMUNICATIONS_CALLS_OPERATIONS

**Description**: Tool to list operations on a Microsoft Teams call. Use when you need to retrieve the status of long-running operations like adding large gallery views, recording, or playing prompts on an active call.
