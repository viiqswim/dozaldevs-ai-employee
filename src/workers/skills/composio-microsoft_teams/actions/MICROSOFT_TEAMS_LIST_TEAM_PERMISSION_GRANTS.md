# MICROSOFT_TEAMS_LIST_TEAM_PERMISSION_GRANTS

**Description**: Tool to list all resource-specific permission grants for a team with support for filtering, pagination, and sorting. Use when you need to identify which Microsoft Entra apps have access to a team and their corresponding permissions.
