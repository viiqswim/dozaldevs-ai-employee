# MICROSOFT_TEAMS_ADD_MEMBER_TO_CHANNEL

**Description**: Tool to add a member to a Microsoft Teams channel. Use this action when you need to grant a user access to a specific channel within a team, or when you need to add an owner to manage the channel.
