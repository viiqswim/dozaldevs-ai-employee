# MICROSOFT_TEAMS_CLEAR_PRESENCE

**Description**: Tool to clear the presence information for a user's application presence session in Microsoft Teams. Use when you need to remove presence information set by an application for the authenticated user. Note: This action can only clear presence for the authenticated user, not for other users.
