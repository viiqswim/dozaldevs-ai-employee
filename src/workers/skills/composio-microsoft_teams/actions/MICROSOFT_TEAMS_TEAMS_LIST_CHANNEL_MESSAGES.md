# MICROSOFT_TEAMS_TEAMS_LIST_CHANNEL_MESSAGES

**Description**: Tool to list messages in a Teams channel when team_id and channel_id are known (no chat_id required). Use this to enumerate channel message history and obtain message_id for follow-on operations like listing replies or getting message details.

| properties | unknown | No |  |
| description | unknown | No |  |
