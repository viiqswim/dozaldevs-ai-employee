# MICROSOFT_TEAMS_LIST_TIME_OFF_REQUESTS

**Description**: Tool to retrieve a list of time off requests from a Microsoft Teams team's schedule. Use when you need to view all time off requests including their status (pending, approved, declined), dates, and associated users.
