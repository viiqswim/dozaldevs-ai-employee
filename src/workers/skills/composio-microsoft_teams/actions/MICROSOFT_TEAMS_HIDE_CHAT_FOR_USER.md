# MICROSOFT_TEAMS_HIDE_CHAT_FOR_USER

**Description**: Tool to hide a Microsoft Teams chat for a specific user. Use when you need to hide a chat from a user's chat list. Note: The chat is automatically unhidden if an action such as sending a message is taken at the chat level.
