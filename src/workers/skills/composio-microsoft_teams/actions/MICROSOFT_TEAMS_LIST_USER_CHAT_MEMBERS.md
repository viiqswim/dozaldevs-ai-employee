# MICROSOFT_TEAMS_LIST_USER_CHAT_MEMBERS

**Description**: Tool to list members of a specific chat for a user in Microsoft Teams. Use when you need to retrieve the members of a specific user's chat, whether it's a one-on-one chat, group chat, or meeting chat.
