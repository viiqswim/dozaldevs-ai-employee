# MICROSOFT_TEAMS_GET_OPEN_SHIFT_CHANGE_REQUEST

**Description**: Tool to retrieve a specific open shift change request from a Microsoft Teams team's schedule. Use when you need to get details about a particular open shift request including its state, sender, and manager actions.
