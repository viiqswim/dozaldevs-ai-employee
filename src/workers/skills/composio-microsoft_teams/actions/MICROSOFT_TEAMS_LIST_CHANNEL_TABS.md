# MICROSOFT_TEAMS_LIST_CHANNEL_TABS

**Description**: Tool to list tabs from a Microsoft Teams channel. Use when you need to retrieve all tabs configured in a specific channel. Note: The Files tab (native to channels) is not returned by this API.
