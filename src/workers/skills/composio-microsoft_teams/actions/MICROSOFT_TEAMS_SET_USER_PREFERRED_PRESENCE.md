# MICROSOFT_TEAMS_SET_USER_PREFERRED_PRESENCE

**Description**: Tool to set the preferred availability and activity status for a user. Use when you need to update a user's presence status in Microsoft Teams. Preferred presence takes effect only when at least one presence session exists for the user.
