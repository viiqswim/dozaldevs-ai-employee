# MICROSOFT_TEAMS_CREATE_SCHEDULE_DAY_NOTE

**Description**: Tool to create a new day note in a team's schedule. Use when you need to add notes or reminders for a specific date in the team schedule. Day notes help communicate important information to team members for a particular day.
