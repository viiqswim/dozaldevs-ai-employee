# MICROSOFT_TEAMS_TEAMS_CREATE_CHAT

**Description**: Creates a new chat; if a 'oneOnOne' chat with the specified members already exists, its details are returned, while 'group' chats are always newly created. IMPORTANT: The authenticated user MUST be included as one of the members.
