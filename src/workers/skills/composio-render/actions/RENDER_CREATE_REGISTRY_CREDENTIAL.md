# RENDER_CREATE_REGISTRY_CREDENTIAL

**Description**: Tool to create a registry credential. Use when you need to add a new container registry credential to your Render account for authenticating with Docker Hub, GitHub, GitLab, Google Artifact Registry, or AWS ECR.
