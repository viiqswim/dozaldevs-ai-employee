# RENDER_UPDATE_ENV_GROUP_ENV_VAR

**Description**: Tool to add or update an environment variable in an environment group. Use when you need to set a new value for an environment variable or create a new one if it doesn't exist. You can either provide a specific value or request a randomly generated value.
