# Platform Shell Tools

Shell tools for the platform service.
Full CLI contract for each tool is in `actions/<tool-id>.md`.

## Available Tools

| Tool | Description |
|------|-------------|
| submit-output | Submit task output (summary and optional draft file) to the platform |
| report-issue | Report a platform issue or error to the issues Slack channel |
| calculate | Evaluate a mathematical expression and return the numeric result |
