# POSTMAN_CREATE_OR_UPDATE_A_SCHEMA_FILE

**Description**: Tool to create or update an API schema file in Postman. Use when you need to add a new schema file or modify an existing one within an API schema. Requires API ID, schema ID, file path, and stringified JSON content.
