# POSTMAN_GET_COLLECTION_ACCESS_KEYS

**Description**: Tool to retrieve all personal and team collection access keys for the authenticated user. Use when you need to list or manage collection access keys. Returns an array of access key objects with their IDs, tokens, status, and associated collection information.
