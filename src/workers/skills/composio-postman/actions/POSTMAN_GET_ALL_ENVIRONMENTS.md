# POSTMAN_GET_ALL_ENVIRONMENTS

**Description**: Tool to get all environments accessible to the authenticated user with optional workspace filtering. Use when you need to list or retrieve environments from Postman. Returns an array of environment objects with their IDs, names, and UIDs.
