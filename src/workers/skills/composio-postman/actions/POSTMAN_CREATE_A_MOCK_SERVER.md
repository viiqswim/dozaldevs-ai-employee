# POSTMAN_CREATE_A_MOCK_SERVER

**Description**: Tool to create a new mock server in a Postman collection. Use when you need to create a mock server to simulate API endpoints for testing or development. Returns the created mock server's details including the mockUrl which can be used to make requests.
