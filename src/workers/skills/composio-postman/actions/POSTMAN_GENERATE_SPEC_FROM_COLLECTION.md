# POSTMAN_GENERATE_SPEC_FROM_COLLECTION

**Description**: Tool to generate an API specification from a Postman collection. Use when you need to create an OpenAPI 3.0 specification from an existing collection. The operation is asynchronous and returns a task ID and polling URL to check the generation status.
