# POSTMAN_UPDATE_PART_OF_A_COLLECTION

**Description**: Tool to update specific collection properties like name, description, authentication, variables, or events. Use when you need to partially update a collection without replacing the entire collection structure. Returns the updated collection information after the changes are applied.
