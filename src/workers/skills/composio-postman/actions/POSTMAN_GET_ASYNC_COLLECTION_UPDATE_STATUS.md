# POSTMAN_GET_ASYNC_COLLECTION_UPDATE_STATUS

**Description**: Tool to get the status of an asynchronous collection update task. Use when you need to check whether a previously initiated async collection update is still processing, has completed successfully, or has failed. The task ID is obtained from PUT /collections/{collectionId} endpoint when using the Prefer: respond-async header.
