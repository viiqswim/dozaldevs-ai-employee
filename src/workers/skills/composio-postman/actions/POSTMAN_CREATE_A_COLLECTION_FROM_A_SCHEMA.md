# POSTMAN_CREATE_A_COLLECTION_FROM_A_SCHEMA

**Description**: Tool to create a collection from a schema and link it to an API with specified relations. Note: This endpoint is deprecated in Postman v10 and higher. Use when you need to generate a collection from an API schema and establish relations like contract tests or documentation.
