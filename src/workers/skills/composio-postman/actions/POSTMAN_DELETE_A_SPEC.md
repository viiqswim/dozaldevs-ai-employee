# POSTMAN_DELETE_A_SPEC

**Description**: Tool to delete an API specification from Postman. Use when you need to permanently remove a specification. On success, returns HTTP 204 No Content response.

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| type | unknown | No |  |
| title | unknown | No |  |

| properties | unknown | No |  |
| description | unknown | No |  |
