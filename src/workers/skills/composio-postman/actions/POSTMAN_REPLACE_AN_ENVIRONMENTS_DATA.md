# POSTMAN_REPLACE_AN_ENVIRONMENTS_DATA

**Description**: Tool to completely replace an environment's data with new variables and values. Use when you need to update an entire environment by replacing all its contents. This operation replaces ALL existing variables with the ones provided in the request.
