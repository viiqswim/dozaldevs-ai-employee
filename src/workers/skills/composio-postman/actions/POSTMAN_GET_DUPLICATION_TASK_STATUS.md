# POSTMAN_GET_DUPLICATION_TASK_STATUS

**Description**: Tool to get the status of a collection duplication task. Use when you need to check whether a previously initiated collection duplication is still processing or has completed. The task ID must first be obtained from the POST /collections/{collectionId}/duplicates endpoint.
