# POSTMAN_GET_ALL_TEAM_USERS

**Description**: Tool to get information about all users on the Postman team. Use when you need to list all team members and their details including roles and join dates. Returns an array of user objects with their IDs, names, usernames, emails, roles, and join timestamps.
