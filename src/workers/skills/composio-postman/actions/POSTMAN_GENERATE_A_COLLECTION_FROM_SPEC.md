# POSTMAN_GENERATE_A_COLLECTION_FROM_SPEC

**Description**: Tool to generate a Postman collection from an OpenAPI 2.0, 3.0, or 3.1 specification. Use when you need to create a collection from an existing API spec. The operation is asynchronous and returns a task ID and polling URL to check the generation status.
