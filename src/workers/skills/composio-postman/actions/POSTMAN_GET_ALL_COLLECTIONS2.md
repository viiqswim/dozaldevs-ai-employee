# POSTMAN_GET_ALL_COLLECTIONS2

**Description**: Tool to get all collections accessible to the authenticated user. Use when you need to retrieve all your collections including subscribed collections. Returns detailed information for each collection including owner, creation/update timestamps, and visibility.
