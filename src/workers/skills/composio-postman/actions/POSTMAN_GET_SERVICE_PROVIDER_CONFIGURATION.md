# POSTMAN_GET_SERVICE_PROVIDER_CONFIGURATION

**Description**: Tool to get Postman's SCIM API service provider configuration information. Use when you need to discover supported SCIM operations, capabilities, and authentication schemes. This endpoint returns configuration details including support for PATCH, bulk operations, filtering, sorting, and ETag handling.
