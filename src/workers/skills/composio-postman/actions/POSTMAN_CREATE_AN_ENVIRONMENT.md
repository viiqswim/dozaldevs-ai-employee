# POSTMAN_CREATE_AN_ENVIRONMENT

**Description**: Tool to create a new environment in a Postman workspace. Use when you need to create a new environment with variables for different settings (development, production, testing, etc.). Returns the created environment's ID, name, and UID upon successful creation.
