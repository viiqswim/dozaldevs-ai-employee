# CANVA_GET_DESIGNS_DESIGNID_EXPORT_FORMATS

**Description**: Lists available file formats for exporting a design. Use this to check which export formats (PDF, JPG, PNG, SVG, PPTX, GIF, MP4) are supported for a specific design.
