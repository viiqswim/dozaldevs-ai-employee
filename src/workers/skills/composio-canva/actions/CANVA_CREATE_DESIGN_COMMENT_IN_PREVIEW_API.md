# CANVA_CREATE_DESIGN_COMMENT_IN_PREVIEW_API

**Description**: Creates a new comment thread on a Canva design. This preview API allows adding text comments to designs for collaboration and feedback. Returns the created comment with ID, author info, and timestamp. Comments can be replied to using the comment reply endpoint.
