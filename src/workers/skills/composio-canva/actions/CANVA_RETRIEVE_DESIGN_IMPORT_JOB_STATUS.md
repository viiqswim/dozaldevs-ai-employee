# CANVA_RETRIEVE_DESIGN_IMPORT_JOB_STATUS

**Description**: Polls for design import job completion status. Use this after CANVA_CREATE_DESIGN_IMPORT_JOB to check the import progress. Repeatedly call this endpoint until a 'success' or 'failed' status is received to get the final imported design ID and metadata.
