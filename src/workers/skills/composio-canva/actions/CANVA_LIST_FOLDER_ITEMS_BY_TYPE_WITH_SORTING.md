# CANVA_LIST_FOLDER_ITEMS_BY_TYPE_WITH_SORTING

**Description**: Lists the items in a folder, including each item's `type`. Folders can contain: - Other folders. - Designs, such as Instagram posts, Presentations, and Documents ([Canva Docs](https://www.canva.com/create/documents/)). - Image assets.
