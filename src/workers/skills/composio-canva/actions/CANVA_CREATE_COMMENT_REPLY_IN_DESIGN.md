# CANVA_CREATE_COMMENT_REPLY_IN_DESIGN

**Description**: This preview API allows replying to comments within a design on Canva, with a limit of 100 replies per comment. Users should note potential unannounced changes, and that preview APIs are not eligible for public integrations.
