# MAILCHIMP_LIST_MESSAGES

**Description**: List all messages from a specific conversation in Mailchimp. NOTE: The Conversations API is deprecated and replaced by Inbox. This endpoint only returns historical conversation data. New campaign replies and Inbox messages are NOT available through this endpoint. Use 'List Conversations' first to obtain valid conversation IDs.
