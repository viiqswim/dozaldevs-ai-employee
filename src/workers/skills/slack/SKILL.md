# Slack Shell Tools

Shell tools for the slack service.
Full CLI contract for each tool is in `actions/<tool-id>.md`.

## Available Tools

| Tool | Description |
|------|-------------|
| post-message | Post a message to a Slack channel |
| read-channels | Read recent messages from one or more Slack channels |
| post-guest-approval | Post a guest-reply approval card to Slack for PM review |
