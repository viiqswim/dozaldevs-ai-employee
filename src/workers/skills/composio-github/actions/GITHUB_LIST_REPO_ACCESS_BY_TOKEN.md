# GITHUB_LIST_REPO_ACCESS_BY_TOKEN

**Description**: Lists repositories in an organization that a fine-grained personal access token has requested access to. Only GitHub Apps can use this endpoint. Requires 'Personal access token requests' organization permissions (read).
