# GITHUB_LIST_TOKEN_ACCESS_REPOSITORIES

**Description**: Lists repositories in an organization that a fine-grained personal access token has access to. Requires organization admin permissions or 'Personal access tokens' organization read permission. Only GitHub Apps can use this endpoint.
