# GITHUB_MANAGE_ACCESS_CONTROL_FOR_ORGANIZATION_CODESPACES

**Description**: Sets the Codespaces access control policy for a GitHub organization, determining which members can use them. This endpoint requires admin permissions on the organization. The API returns HTTP 204 No Content on success.
