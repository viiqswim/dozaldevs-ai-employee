# GITHUB_GET_GITIGNORE_TEMPLATE

**Description**: Retrieves a specific .gitignore template from GitHub by its name, which must be an existing template in GitHub's collection.

|-----------|------|----------|-------------|
| type | unknown | No |  |
| title | unknown | No |  |

| description | unknown | No |  |
