# GITHUB_LOOKUP_ENTERPRISE_ADMINISTRATOR_INVITATION

**Description**: Tool to look up a pending enterprise administrator invitation by invitee, enterprise, and role. Use when you need to check if a specific user has a pending invitation to become an enterprise administrator.
