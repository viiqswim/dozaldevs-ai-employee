# GITHUB_REQUEST_REVIEWERS_FOR_A_PULL_REQUEST

**Description**: Adds reviewers to a specific pull request identified by owner, repo, and pull_number. This is NOT a search action - it operates on an already-known PR number. Requires `owner`, `repo`, `pull_number`, and at least one of `reviewers` or `team_reviewers`.

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| type | unknown | No |  |

| required | unknown | No |  |
| properties | unknown | No |  |
| description | unknown | No |  |
