# GITHUB_LIST_ORG_PROJECT_VIEW_ITEMS

**Description**: Tool to list items for an organization project view. Use when you need to retrieve all items from a specific view within a GitHub organization's Projects V2 project. Returns a list of project items with their content, metadata, and field values.
