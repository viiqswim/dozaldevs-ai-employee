# GITHUB_LIST_ORG_REPOS_WITH_GITHUB_ACTIONS_ENABLED

**Description**: Lists the selected repositories that are enabled for GitHub Actions in an organization. This endpoint only returns results when the organization's 'enabled_repositories' permission policy is configured to 'selected'. Requires admin:org scope and organization admin permissions.
