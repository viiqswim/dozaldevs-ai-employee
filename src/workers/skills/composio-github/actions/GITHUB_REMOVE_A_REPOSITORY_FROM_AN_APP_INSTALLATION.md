# GITHUB_REMOVE_A_REPOSITORY_FROM_AN_APP_INSTALLATION

**Description**: Removes a repository from a GitHub App installation for the authenticated user, given a valid `installation_id` and the `repository_id` of a repository currently linked to that installation; this operation is idempotent.

| title | unknown | No |  |

| description | unknown | No |  |
