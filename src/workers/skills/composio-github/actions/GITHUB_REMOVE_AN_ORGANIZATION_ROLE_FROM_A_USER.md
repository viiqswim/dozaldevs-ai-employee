# GITHUB_REMOVE_AN_ORGANIZATION_ROLE_FROM_A_USER

**Description**: Removes a custom organization role from a user within a GitHub organization, provided the organization exists, the user is a member, and the role_id corresponds to a valid custom role in that organization.
