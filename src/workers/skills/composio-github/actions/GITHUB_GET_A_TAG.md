# GITHUB_GET_A_TAG

**Description**: Retrieves detailed information for a specific Git tag object from a GitHub repository, using the SHA of the tag object itself.

|-----------|------|----------|-------------|
| type | unknown | No |  |
| title | unknown | No |  |
| required | unknown | No |  |
| properties | unknown | No |  |
| description | unknown | No |  |
