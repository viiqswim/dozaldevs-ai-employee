# GITHUB_SET_INTERACTION_RESTRICTIONS_FOR_A_REPOSITORY

**Description**: Temporarily limits which GitHub users (e.g., existing_users, contributors_only) can interact (comment, open issues, create pull requests) in a repository for a specified duration.
