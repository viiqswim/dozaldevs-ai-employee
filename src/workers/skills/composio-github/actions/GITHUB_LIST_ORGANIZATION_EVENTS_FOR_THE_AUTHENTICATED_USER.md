# GITHUB_LIST_ORGANIZATION_EVENTS_FOR_THE_AUTHENTICATED_USER

**Description**: Lists public events in a specified GitHub organization for the authenticated user's organization dashboard. Returns activities like repository pushes, pull requests, issues, forks, and other public events within the organization. The authenticated user must be a member of the organization to see events.
