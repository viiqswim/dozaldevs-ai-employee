# GITHUB_SET_USER_INTERACTION_LIMIT_GRAPH_QL

**Description**: Tool to set interaction limits on a user's public repositories using GraphQL. Use when you need to temporarily restrict who can comment, open issues, or create pull requests. Limits apply to the authenticated user or a specified user.
