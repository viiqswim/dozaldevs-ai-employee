# GITHUB_REMOVE_SELECTED_REPO_FROM_ORG_VARIABLE

**Description**: Removes a repository's access to an organization variable that has 'selected' visibility. This endpoint removes a specific repository from an organization variable's access list. The variable must have its visibility set to 'selected' for this operation to work. Returns a 409 Conflict error if the variable visibility is not 'selected'.
