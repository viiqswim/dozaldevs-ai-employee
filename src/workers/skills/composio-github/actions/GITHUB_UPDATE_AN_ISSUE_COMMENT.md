# GITHUB_UPDATE_AN_ISSUE_COMMENT

**Description**: Updates an existing comment on an issue or pull request within a specified repository. You can only update comments that the authenticated user has permission to edit. The comment_id can be obtained from the list_issue_comments or get_an_issue_comment actions.

| properties | unknown | No |  |
| description | unknown | No |  |
