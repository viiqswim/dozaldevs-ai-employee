# GITHUB_SUBMIT_A_REVIEW_FOR_A_PULL_REQUEST

**Description**: Finalizes a pending pull request review (identified by `review_id`) with a required `event` (APPROVE, REQUEST_CHANGES, COMMENT) and an optional `body`.
