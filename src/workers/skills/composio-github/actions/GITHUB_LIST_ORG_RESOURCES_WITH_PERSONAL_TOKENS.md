# GITHUB_LIST_ORG_RESOURCES_WITH_PERSONAL_TOKENS

**Description**: Lists requests from organization members to access organization resources with fine-grained personal access tokens (pending, approved, or denied). Note: This endpoint requires GitHub App authentication with 'organization_personal_access_token_requests:read' permissions; it cannot be used with personal access tokens.
