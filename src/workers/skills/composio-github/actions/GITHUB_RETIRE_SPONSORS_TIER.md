# GITHUB_RETIRE_SPONSORS_TIER

**Description**: Tool to retire a published payment tier from your GitHub Sponsors profile using GraphQL. Use when you need to prevent a tier from being used for new sponsorships. Existing sponsors at this tier remain unaffected.

| Parameter | Type | Required | Description |

| type | unknown | No |  |

| required | unknown | No |  |
| properties | unknown | No |  |
