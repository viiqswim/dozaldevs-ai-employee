# GITHUB_SET_REPO_ALLOWED_ACTIONS

**Description**: Sets allowed GitHub Actions and reusable workflows for a repository, managing permissions for GitHub-owned, verified creator, or specific pattern-matched actions/workflows (note: `patterns_allowed` applies only to public repositories).
