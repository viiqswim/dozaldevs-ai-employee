# GITHUB_LIST_SELECTED_REPOSITORIES_FOR_SECRET_ACCESS

**Description**: Lists repositories within a specified organization that have been granted access to a particular Dependabot secret. Note: This endpoint only returns results when the secret's visibility is set to 'selected'. Secrets with visibility 'all' or 'private' do not have specific repository selections. Requires 'admin:org' scope and the user must have admin access to the organization.
