# GITHUB_LIST_RUNNER_APPLICATIONS_FOR_AN_ORGANIZATION

**Description**: Lists downloadable GitHub Actions runner application binaries for setting up self-hosted runners in a GitHub organization. Returns download URLs and checksums for various OS/architecture combinations (linux/win/osx on x64/arm/arm64). Requires admin:org scope or fine-grained 'Self-hosted runners' read permission.
