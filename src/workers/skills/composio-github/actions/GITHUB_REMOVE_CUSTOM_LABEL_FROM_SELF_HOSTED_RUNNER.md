# GITHUB_REMOVE_CUSTOM_LABEL_FROM_SELF_HOSTED_RUNNER

**Description**: Removes a currently assigned custom label (`name`) from a self-hosted runner (`runner_id`) in an organization (`org`). This action allows the organization to manage the labels associated with their runners effectively.
